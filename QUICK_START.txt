## 📋 SIMPLE SYSTEM - SETUP (5 Minutes)

### Step 1: Add Student Photos
1. Open the `images` folder (just created)
2. Add student photos with names as filenames:
   ```
   images/
   ├── JohnDoe.jpg
   ├── JaneSmith.jpg
   ├── MikeBrown.jpg
   └── SarahJones.jpg
   ```

**Important:**
- Use clear, front-facing photos
- One face per image
- Good lighting
- Supported: .jpg, .png, .jpeg

### Step 2: Run the System
```bash
python simple_face_recognition.py
```

### Step 3: Use the System
1. System loads faces (first time takes 10-30 seconds)
2. Camera opens automatically
3. Stand in front of camera
4. System recognizes and marks attendance automatically
5. Press 'Q' to quit

### What You'll See:

## 🐛 TROUBLESHOOTING

### "No images found in images folder"
**Fix:** Add at least one .jpg image to `images` folder

### "No face found in: image.jpg"
**Fix:** 
- Use front-facing photo
- Ensure face is clearly visible
- Good lighting
- Only one face in image

### "Cannot access camera"
**Fix:**
- Close Zoom, Skype, Teams
- Check Windows Settings → Privacy → Camera
- Restart computer

### face-recognition install failed
**Already installed!** ✅ We just installed it successfully
